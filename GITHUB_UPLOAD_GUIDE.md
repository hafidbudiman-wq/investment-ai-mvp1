# Upload ke GitHub – Cara Termudah

## Metode website GitHub

1. Masuk ke GitHub.
2. Klik **New repository**.
3. Repository name: `investment-ai-stock-analysis`.
4. Pilih **Private** selama tahap pengembangan.
5. Jangan centang Add README karena README sudah tersedia.
6. Klik **Create repository**.
7. Pilih **uploading an existing file**.
8. Ekstrak ZIP project ini di laptop.
9. Drag seluruh isi folder `investment-ai-stock-analysis` ke halaman upload GitHub.
10. Isi commit message: `Initial InvestAI MVP 1`.
11. Klik **Commit changes**.

Jangan meng-upload file `.env`. Gunakan `.env.example` sebagai contoh konfigurasi.
